IO_demo 文件夹内为STM32 F1驱动程序，主要负责遥控信号接收，电机PWM驱动等底层功能
FMU_demo 文件夹内为STM32 F4驱动程序，负责数据处理，存储，导航等上层应用

创建工程方式为MDK5+CUBEMX ST-LINK调试
